# Cập nhật Chinese Mission Việt 2.3 bằng GitHub Desktop

1. Giải nén gói cập nhật 2.3.
2. Mở GitHub Desktop và chọn repository `chinese-mission-viet`.
3. Chọn **Repository → Show in Explorer**.
4. Mở thư mục `chinese-mission-viet` trong gói cập nhật.
5. Sao chép toàn bộ nội dung vào thư mục repository hiện tại.
6. Chọn **Replace the files in the destination** khi Windows hỏi.
7. Quay lại GitHub Desktop, nhập Summary: `Nâng cấp chất lượng bài học 2.3`.
8. Chọn **Commit to main** rồi **Push origin**.
9. Vercel sẽ tự động triển khai. Không cần chạy lại SQL Supabase và không cần sửa biến môi trường.

Sau khi triển khai, vào một bài bất kỳ. Bên dưới phần “Câu mẫu cần luyện” sẽ có khối “Kiến thức mở rộng đã chuẩn hóa”.
